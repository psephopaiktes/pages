Cool Bookmarklets

ブラウザをより便利に利用できるようになるブックマークレットの紹介です。ブックマークレットとは、ブラウザのブックマークに登録しておき、特定のWebサイト上でクリックすることで利用できるプログラムです。例えば見ているWebサイトを素早くSNSでシェアしたり、URLをコピーしたりできます。Chrome拡張機能と違ってメモリを専有したりせず、気軽に利用できます。ブックマークバー上にフォルダを作ってまとめておくと、たくさん登録しても邪魔になりません。

使い方

  1. 利用したいブックマークレットのボタンをブラウザのブックマークバーにドラッグ・アンド・ドロップして、ブックマークとして登録しておきます。
  2. 対象となるWebサイトを開き、そのサイト上で登録しておいたブックマークレットをクリックします。

本ページ内で直接公開しているものは私が作成したブックマークレットです。SNSでのシェアなどは公式や外部の方が作られたものなどいろいろありますが、再配布を禁止しているところもあるので外部リンクで紹介しておきます。

ブックマークレット一覧

    サイトのタイトルをコピー

    開いているWebサイトのタイトル( titleタグに指定されているテキスト )をクリップボードにコピーします。通常サイトのタイトルはソースを確認しないとコピーできないので、地味に便利です。

    copy title
    このボタンをブックマークバーにドラッグします

    URLをHTML形式でコピー

    開いているWebサイトのURLとタイトルを、HTMLのaタグ形式でクリップボードにコピーします。

    copy html link
    このボタンをブックマークバーにドラッグします

    URLをMarkdown形式でコピー

    開いているWebサイトのURLを、Markdwonのリンク形式でクリップボードにコピーします。

    copy md link
    このボタンをブックマークバーにドラッグします

    日本語を含むURLをエンコードせずにコピー

    開いているWebサイトのURLが日本語を含む場合、そのまま開くとエンコードされて無駄に長いURLになってしまいます。このブックマークレットなら日本語のままURLをコピーできます。

    copy link as Japanese
    このボタンをブックマークバーにドラッグします

    Google検索の期間指定

    Google検索の結果ページで実行すると、対象のサイトを3年間以内のものに限定します。

    search in 3years
    このボタンをブックマークバーにドラッグします

    サイトをGoogle翻訳

    日本語以外のWebページで実行すると、対象のサイトをGoogle翻訳で日本語に翻訳します。

    translate JP
    このボタンをブックマークバーにドラッグします

    サイト内(ドメイン内)検索

    対象となるWebサイト全体の中(ドメイン内)でテキスト検索します。よくあるブックマークレットですが、挙動が微妙なものばかりなので自作しました。

    search in domain
    このボタンをブックマークバーにドラッグします

    スマホファーストビューチェッカー

    スマホで見た時にファーストビューがどのような表示になるか、ブラウザの枠付きで確認できます。とりあえずiPhone7だけ対応です。

    first-view
    このボタンをブックマークバーにドラッグします

    説明

    事前にChromeのモバイルプレビューモードをONにします(⌘⌥I→⌘⇧M)。iPhone7を選択します。

    クトゥルフ神話TRPG用ブックマークレット

    かなりニッチなブックマークレットです。自分用に作成しました。クトゥルフ神話TPRGでキャラクターシートを作成する際に便利です。

    CoC Dice Roller
    このボタンをブックマークバーにドラッグします

    説明

    キャラシ作成サイトクトゥルフPC 作成ツールをより便利に使用できるブックマークレットです。

    • 一括で能力値ダイスをふって、自動入力
    • 作成したキャラを読みやすい形式でキャラクターシートに出力
    する機能を追加します。

    使い方

    1. クトゥルフPC 作成ツールにアクセスします。
    2. 事前に登録しておいたこのブックマークレット ( 保存したブックマーク ) をクリックします。
    3. ページの上部にボタンが2つ出現します。

    DiceRoll ボタン

    • 基本能力値をランダムに決定し、自動で入力します。
    • ボタンを押すたびにダイスがふり直されます。
    • すでに能力が入力されている場合でも上書きしてしまうのでご注意ください。

    キャラクターシート出力

    • 入力されている能力値から簡易的なキャラクターシートを出力します
    • ボタンを押すと、クリップボードにキャラクターシートがコピーされます
    • メモ帳などにCtrl+Vでペーストしてください。
